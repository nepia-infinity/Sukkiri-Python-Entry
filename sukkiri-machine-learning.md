# 機械学習（Machien Learning）
- ### 教師あり学習（Supervised Learning）
    - ex. 画像から動物名を予測するための法則をコンピューターに自動で探させることが出来る

    - **大量の画像データが必要**
    - 入力データと答えのデータのペアが必要で、教師データ（Training Data）という
        - 回帰（Regression）
        - 分類（classification）


- ### 教師なし学習（Unsupervised Learning）
    - 実態としては、答えなし学習に近い
    - クラスタリング（Clustering）
    - 次元削減（Dimentionary Reduction）
    - **似ているデータ同士をグループ分けする**
